## 1. Mermaid Diagram Features (WS1 + WS2)

### When to Use
You need visual representations of:
- Drug mechanisms of action
- Signaling pathways
- Metabolic cycles
- Process flowcharts
- Timelines

### How to Use

**Request a Diagram:**
```
Draw a flowchart of the RAAS pathway
Create a diagram showing warfarin mechanism
Visualize the drug approval process
Map out the CYP450 enzyme system
```

**If Diagram Fails to Render:**
1. Click the **Refresh** button on the error box
2. First click: Regex auto-fix (instant)
3. If still broken: AI repair triggers automatically (5-10 seconds)
4. Watch for amber "Repairing..." spinner

**What You'll See:**
- ✅ Success: Interactive SVG diagram with zoom/pan
- ⚠️ Error: Raw code shown with "Repair" button
- 🔧 Repairing: Amber spinner → "Fixed!" toast

## 2. Manuscript Export (WS3)

### When to Use
You want to save a conversation as:
- Professional report
- Academic manuscript
- Structured document with sections

### How to Use

1. **Open any conversation** with AI responses
2. **Click the Download icon** (📥) on any assistant message
3. **Select export format:**
   - **Export as Chat** → Raw conversation (plain)
   - **Export as Report** → Executive summary format
   - **Export as Manuscript** → Academic IMRAD structure

**What You'll Get:**
- DOCX file with:
  - Title page (Benchside branding)
  - Table of Contents (auto-generated in Word)
  - Numbered sections (H1, H2, H3)
  - Page numbers in footer
  - "Generated by Benchside" watermark

**File Name:** `manuscript_[conversation_id]_[style].docx`

## 3. Citation Manager (WS4)

### When to Use
AI response includes:
- DOI references (e.g., `10.1038/s41586-021-03819-2`)
- PMID references (e.g., `PMID: 34265844`)
- Academic citations

### How to Use

**Automatic Detection:**
- Citations are auto-detected in AI responses
- Amber "References" badge appears below message

**Interact with Citations:**
1. **Click "3 References"** badge to expand
2. **Per-citation actions:**
   - Click **APA** → Copies APA format to clipboard
   - Click **BibTeX** → Copies BibTeX entry
   - Click title → Opens DOI/PubMed link
3. **Bulk actions:**
   - Click **Download .bib** → Downloads all citations as `.bib` file

**What You'll See:**

## 4. PubMed Search (WS5)

### When to Use
You need to find:
- Recent research articles
- Clinical trial results
- Drug mechanism studies
- Disease pathophysiology papers

### How to Use

**Trigger Search:**
```
/pubmed SGLT2 inhibitors diabetes
/pubmed CAR-T therapy lymphoma
/pubmed GLP-1 agonists weight loss
```

**What You'll See:**
- Card grid (2 columns on desktop, 1 on mobile)
- Each card shows:
  - **Title** (clickable → PubMed)
  - **Authors** (truncated with "et al.")
  - **Journal + Year**
  - **Abstract preview** (2 lines)
- **Actions per card:**
  - **Show abstract** → Expands full abstract
  - **Copy APA** → Copies citation

**Footer:**
- "Source: PubMed® via NCBI E-utilities"
- "View on PubMed" link (opens full search)

## Troubleshooting

### Mermaid Diagram Won't Render
1. Click **Refresh** button
2. Wait for AI repair (amber spinner)
3. If still broken: Ask AI to "simplify the diagram"

### Export Returns 404
- **Cause**: Invalid conversation ID
- **Fix**: Make sure you're in a valid conversation (not "chat" placeholder)

### PubMed Search Returns No Results
- Try broader terms: "diabetes" instead of "type 2 diabetes mellitus"
- Check spelling
- Use MeSH terms if known

### DDI Shows "Could Not Resolve Drug"
- Try generic name instead of brand
- Use common spelling (e.g., "acetaminophen" vs "paracetamol")

### Slide/Doc Generation Stuck
- Check browser console for errors
- Refresh page and try again
- Large outlines (>20 slides/sections) may timeout
